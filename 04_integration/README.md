## The Integration Principle

Every output from every model in this repo has a named
consumer. Nothing is generated for its own sake.

The LHS valuation output feeds the Ventures investment
committee decision framework. The macro stress output
feeds the Ventures LP reporting section. The historical
simulation output feeds the quarterly LP risk report
template in Ventures and the Series A and B pitchbooks.

This repo does not stand alone. It is the quantitative
engine that makes the other two repos institutionally
credible. Without this repo, Ventures is a thesis.
With this repo, Ventures is a risk-managed capital
deployment framework.
